概览

本版本前端框架组件基于Bootstrap3选型，采用扁平化的风格，以库/框架选型为主，以满足小型项目的快速开发。

技术准备

需要掌握HTML、CSS、JavaScript等知识，以便开展工作。

标准规范

Bootstrap 使用到的某些 HTML 元素和 CSS 属性需要将页面设置为 HTML5 文档类型。在你项目中的每个页面都要参照下面的格式进行设置。

<!DOCTYPE html>
<html lang="zh-CN">
  ...
</html>

安装方式

直接在HTML文件中引入相应的 CSS and JS 文件即可。

<!DOCTYPE html>
<html>
    <head>
    <title>demo</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Bootstrap -->
    <link href="../bootstrap.min.css" rel="stylesheet" media="screen">
    </head>
    <body>
    <!-- jquery -->
    <script src="../jquery.js"></script>
    <!-- Bootstrap -->
    <script src="../bootstrap.min.js"></script>
    </body>
</html>

设置成功! 加入这三个文件, 你就可以开始用Bootstrap开发任何网站和应用程序了。
